# Social network - Sarafan - https://sarafan-ls.herokuapp.com/auth

## ***Technologies used:***

* **Spring Boot Rest Application + Vue.js**
* **PostgreSQL**
* **Jackson**
* **oAuth2**
* **Spring Security**
* **Webpack**
* **WebSocket (SockJS + Stomp)**
* **Vuetify (Material design)**
* **JsonView Serialization**
* **Vuex**
* **Vue Router**
* **Open Graph Protocol**
